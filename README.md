# E-Commerce Sales Analysis

## Project Overview
Analyzed e-commerce sales data using SQL and Power BI to identify sales trends, top-performing categories, customers, and regions.

## Tools Used
- SQL (MySQL)
- Power BI
- Excel

## Key Analysis
- Total Sales Analysis
- Regional Sales Analysis
- Category Analysis
- Top Customers
- Product Performance

## SQL Concepts Used
- GROUP BY
- ORDER BY
- Aggregate Functions
- LIMIT
- DISTINCT
